# cheatsheets

Cheatsheets requires Mkdocs and the material theme to run locally and publish

### Installing mkdocs
    pip install mkdocs
    mkdocs --version
  
Note: MkDocs supports Python versions 2.7, 3.3, 3.4, 3.5 and pypy.
    
### Installing the material theme
    pip install mkdocs-material


### To run
    mkdocs serve
